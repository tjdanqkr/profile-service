== 공통 응답 형식

모든 API는 아래와 같은 공통 응답 포맷을 사용합니다.

[source,json]
----
{
  "success": true,
  "statusCode": 200,
  "data": { ... },
  "message": "성공 메시지",
  "timestamp": "2025-04-28T12:00:00"
}
----

- `success` : 요청 성공 여부
- `statusCode` : HTTP 응답 코드
- `data` : API별 결과 데이터
- `message` : 에러 발생 시 메시지
- `timestamp` : 요청 처리 시간

== 에러 코드

|===
| 에러 코드 | 설명

| 400 | 잘못된 요청 (Validation 실패 등)
| 404 | 리소스를 찾을 수 없음 (ex. 프로필 없음)
| 409 | 비즈니스 로직 충돌 (ex. 중복 결제 등)
| 500 | 서버 내부 오류
|===
